# 🎯 Number Guessing Game (Java)

## 📌 Description
This is a console-based Java application where the user tries to guess a randomly generated number between 1 and 100.

---

## 🚀 Features
- Random number generation (1–100)
- User input handling
- Hints: Too High / Too Low
- Tracks number of attempts
- Simple CLI-based interaction

---

## 🛠️ Technologies Used
- Java
- Random Class
- Loops and Conditional Statements

---

## ▶️ How to Run
1. Open in Eclipse or any Java IDE
2. Run `NumberGuessingGame.java`
3. Enter guesses until correct
